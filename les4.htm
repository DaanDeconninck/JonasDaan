Les 4

In onze vierde workshop rondden we het deel ICT af. Hierin leerden we werken met PHP, dit is een scripttaal die op webservers dynamische webpagina’s kan creëren. Met deze taal kunnen we onze webpagina verbinden met een database en dus gegevens opslaan. Zo moesten we in deze les een formulier opstellen met PHP dat dient voor ouders die hun kind willen inschrijven voor een zomerkamp. PHP valt makkelijk te combineren met HTML waardoor we deze makkelijk kunnen verbinden met elkaar. Hierdoor zal het formulier een vaste structuur krijgen.
